Горячие клавиши
Sublime Text

Выделение

  • ctrl+a выделить всё содержимое
  • ctrl+l выделить всю строчку
  • ctrl+d выделить слово
  • ctrl+shift+j выделить абзац
  • ctrl+shift+a выделить содержимое тега
  • ctrl+shift+m выделить содержимое в скобках

Текст

  • ctrl+shift+enter вставить строку перед кареткой
  • ctrl+enter вставить строку после каретки
  • ctrl+shift+k удалить строку
  • ctrl+delete удалить слово после каретки
  • ctrl+backspace удалить слово перед кареткой

Комментарии

  • ctrl+/ Закомментировать
  • ctrl+shift+/ Закомментировать блок

Управление каретками

  • ctrl+alt+up добавить каретку в верхнюю строчку
  • ctrl+alt+down добавить каретку в нижнюю строчку
  • ctrl+shift+l добавить каретки в конец строк

Работа со строками

  • ctrl+] отступ
  • ctrl+[ обратный отступ
  • ctrl+j объединение строк

Работа с тегами

  • alt+. закрыть открытый тег
  • ctrl+shift+a выделить содержимое тега

Поиск/замена

  • ctrl+f поиск
  • f3 следующий результат поиск
  • shift+f3 предыдущий результат поиска
  • ctrl+i быстрый поиск введенного запроса
  • ctrl+shift+f поиск по файлам
  • ctrl+h заменить

Переходы

  • ctrl+p переход к файлу
  • ctrl+pagedown следующий файл
  • shift+pageup предыдущий файл
